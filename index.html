microBean™ Scope

The microBean™ Scope project provides classes and interfaces assisting with implementing dependency injection scopes.

Status

This project is currently experimental, in a pre-alpha state, and unsuitable for production use.

Compatibility

Until further notice, this project's APIs are subject to frequent backwards-incompatible signature and behavior changes, regardless of project version and without notice.

Requirements

microBean™ Scope requires a Java runtime of version 23 or higher.

Installation

microBean™ Scope is available on Maven Central. Include microBean™ Scope as a Maven dependency:

<dependency>
  <groupId>org.microbean</groupId>
  <artifactId>microbean-scope</artifactId>
  <!-- Always check https://search.maven.org/artifact/org.microbean/microbean-scope for up-to-date available versions. -->
  <version>0.0.5</version>
</dependency>

Documentation

Full documentation is available at microbean.github.io/microbean-scope.
